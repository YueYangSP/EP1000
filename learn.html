CSS

CSS stands for Cascading Style Sheets, it is a method used to describe the presentaion of a document written in languages such as HTML (HyperText Markup Language) or XML (eXtendable Markup Language) enables programmers to change the look, feel and overall presentaion of a website, allowing developers to customise every aspect of their websites to their liking. The website template used as first draft for this site is detailed in this video by Jake Wright


Examples of CSS being used to great effect:

  • Blue coloured line of text
  • Green coloured, large font line of text
  • Link to google.com


Javascript

Javascript is used in website development to aid or enable the use of:

  • Dynamically Updating Content
  • Multimedia Control
  • Animate Images
  • Others unmentioned
    • JavaScript is a scripting language that is plenty useful in supplementing the shortfalls of both HTML and CSS, completing websites that are otherwise impossible to make.
