HTML

HTML stands for HyperText Markup Language,it is the standard markup language for documents designed to be displayed in a web browser. HTML is used to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items.
HTML elements are delineated by tags, written using angle brackets. Tags such as <img> and <input> directly introduce content into the page. Otherwise, tags such as <p> surround and provide information about document text and may include other tags as sub-elements.
Browsers do not display the HTML tags but use them to interpret the content of the page.


CSS

CSS stands for Cascading Style Sheets, it is a method used to describe the presentaion of a document written in languages such as HTML (HyperText Markup Language) or XML (eXtendable Markup Language) enables programmers to change the look, feel and overall presentaion of a website, allowing developers to customise every aspect of their websites to their liking. The website template used as first draft for this site is detailed in this video by Jake Wright.
Massively helpful reference website for CSS: w3schools.com

Examples of CSS being used to modify look of text:

  • Blue coloured line of text
  • Green coloured, larger, bolded font line of text
  • Red coloured, italic line of text
  • Yellow coloured, Roboto font line of text
Examples of CSS being used to modify look of link text:
  • Plain Link to Twitter
  • Purple Coloured link to Google
  • Yellow coloured italic link to Youtube
