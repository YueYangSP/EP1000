Why EP1000



Learning HTML

I have always been interested in programming, be it C++, python, javascript or html. Since this module mandates the use of html, i used it as an opportunity to learn and use html for website coding.



Opportunity to fabricate own product

This module provides me with the materials and knowledge to fabricate a product of my own choosing.
I decided on making a reading light with the following features:

  • variable red light intensity (knob or slider)
  • variable white light intensity (knob or slider)
  • 3 swivel joints, 2 rotary joints
  • main power button
  • illuminated buttons (very soft lighting) (for night use)
